# Pattern Library

Notation:

- `D`: don, center hit
- `K`: ka, rim hit
- `R`: roll/rapid section marker
- `-`: rest

Patterns are written on a 16th-note grid per 4/4 bar. Spaces separate beats.

## Easy

Use stable phrases that cover the core groove. Easy is simpler, not necessarily empty; dense songs can still have many Easy notes when the music's basic drum pulse is active.

```text
D--- ---- D--- ----
D--- K--- D--- K---
D--- ---- ---- K---
D--- ---- D--- K---
D-D- D-D- D-D- D-D-
D-D- K-D- D-D- K-D-
```

Rules:

- Prefer downbeats.
- Avoid more than two notes per beat.
- Keep `K` rare and meaningful.
- Use repeated bars in verse/intro to build confidence.
- For drum-dense songs, keep the basic pulse but simplify color and variation.

## Normal

Use readable 8th-note motion, simple color, and light variation:

```text
D--- D--- K--- D---
D-K- D-K- D--- K---
D--- DK-- D--- DK--
D-D- K--- D-D- K---
D--- D-K- D--- K-D-
```

Rules:

- Use one or two motifs per section.
- Repeat chorus motifs with small variations every two or four bars.
- Place `K` on answers/off-beats, not randomly.
- Insert rest bars after busy fills.

## Hard

Use short 16th bursts, syncopation, and motif variation:

```text
D-DK D-DK D-K- D---
DKDK D-KD DK-- D---
DDKD KDDK DDKK DKDK
D-DK DKDD K-DK DDK-
DD-- K-DD D-KD K---
D-KD D-KD DKDK D---
```

Rules:

- Bursts should answer clear musical energy.
- Use repeated hand-feel patterns; avoid noise-like note placement.
- Do not exceed four dense bars without relief unless the song clearly sustains peak energy.
- Prefer variation through phrase endings rather than changing every bar.

## Section Recipes

Intro:

```text
D--- ---- D--- ----
D--- ---- K--- ----
```

Verse:

```text
D--- ---- D--- K---
D--- D--- ---- K---
```

Pre-chorus/build:

```text
D--- D--- K--- D---
D-K- D-K- D--- K---
```

Chorus/drop:

```text
D-DK D-DK D-K- D---
DKDK D-KD DK-- D---
```

Bridge/break:

```text
D--- ---- ---- K---
---- D--- ---- D---
```

Fill:

```text
D-DK DKDD K-DK DDK-
DDKD KDDK DDKK DKDK
```

## Pattern Selection Heuristic

Choose the pattern that:

1. places `D` on the strongest accents
2. matches the target local density
3. continues or varies the previous motif
4. gives the section a distinct feel
5. avoids sudden unmotivated hand/color complexity
